ThermoProps
===========
This section documents the ``ThermoProps`` class encoded in the
``thermo.py`` file.  This class contains inherits the Helmholtz
class and uses its functionality to solve for thermodynamic properties
cuch as gas pressure, temperature, density, isochoric and isobaric 
specific heats, entropy, enthalpy, internal energy, and speed of sound.
